# 📊 Sales Report Dashboard — Power BI

## 🚀 Visão Geral do Projeto

Este projeto apresenta um **dashboard interativo desenvolvido no Power BI** para análise estratégica de vendas, utilizando **parâmetros dinâmicos**, métricas avançadas e visualizações orientadas ao **storytelling**.

O relatório foi construído com base no desafio do módulo anterior, incorporando recursos que permitem ao usuário explorar os dados de forma intuitiva, personalizada e visualmente atrativa.

---

## 🎯 Objetivo

Desenvolver um painel analítico que permita:

- 📈 Acompanhar o desempenho de vendas ao longo do tempo  
- 🔄 Alternar métricas e categorias por meio de parâmetros  
- 🧭 Explorar os dados de forma interativa  
- 📖 Contar uma história clara para apresentação ao cliente  

---

## 🧩 Estrutura do Dashboard

O relatório é composto por:

### ✅ Indicadores Principais (KPIs)
- Total de Vendas  
- Unidades Vendidas  
- Soma de Descontos  
- Custo dos Produtos (COGS)

### ✅ Análises Visuais
- 📆 Vendas por Período (Linha)
- 🏷️ Vendas por Segmento (Barras)
- 📦 Vendas por Produto
- 🌍 Vendas por País (Treemap)
- 📊 Tabela Analítica por Trimestre

### ✅ Filtros Interativos
- Segmentação por data
- Parâmetros dinâmicos
- Navegação entre páginas

---

## 🔧 Parâmetros Implementados

Foram criados parâmetros conforme as diretrizes do curso:

### 🔹 Primeira Visão — Categorias
Permite alternar análises entre:
- Segmentos
- Produtos
- Países
- Outras categorias

### 🔹 Segunda Visão — Valores
Permite comparar métricas como:
- Sales
- Profit
- Discount
- Units Sold

Esses parâmetros tornam o dashboard flexível e altamente explorável.

---

## 🎨 Design e Storytelling

O layout segue a identidade visual do relatório original, utilizando:

- Paleta de cores padronizada
- Organização hierárquica
- Destaque para indicadores-chave
- Fluxo visual lógico

A página foi estruturada como uma **apresentação ao cliente**, priorizando clareza, impacto visual e narrativa dos dados.
